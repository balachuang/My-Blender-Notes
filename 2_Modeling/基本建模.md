### 基本建模

- 選單 Edit >> Preferences >> Keymap >> Pie Menu on Dag: 打勾後, 按 Tab 不放再移動滑鼠可以顥示圓餅選單.

### 建模常用快速鍵
- 環選: Atl + Click
- 反選: Ctrl + I (點, 線, 面 都適用)
- 多選: Shift + Click (點, 線, 面 都適用)
- 選最短路徑: Ctrl + Click (點, 線, 面 都適用)
- 選相連物件: L
- 切換透明 Mode: Alt + Z
- 選邊之後建面: Alt + F: (Fill)
- 插面: I (Inset)
- 擠出: E (Extrude)
- Poly Build --> 很難用
  - 滑鼠選邊直接拉: 擠出新的四角面
  - Ctrl + 滑鼠選邊直接拉: 擠出新的三角面
  - Shift + 滑鼠選面: 刪除面
  - Ctrl + 滑鼠 Click: 新增點
  - 選多點按 F: 填充面
  - Ctrl + 滑鼠選角頂點直接拉: 建出新的四角面
